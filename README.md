# Premium Bricks — Paywall

## Архітектура
**MVVM** — очевидний вибір для одноекранного SwiftUI-застосунку:

- `SubscriptionPlan` (Model) — enum з двома планами, цінами та обчислюваними рядками
- `PaywallViewModel` (ViewModel), вся бізнес-логіка: purchase, restore, відкриття посилань, `PurchaseState` enum для станів
- `PaywallView` + (View) декларативний UI без логіки

## Що зрозуміло з макету

- Два плани: **Yearly** (BEST VALUE, зелений бейдж) і **Monthly**
- При виборі Monthly кнопка → **"Start Free Trial"**; при виборі Yearly → **"Continue"**

## Припущення

- Початковий стан — Monthly вибраний за замовчуванням (як у Paywall 1 на макеті)
- StoreKit не підключений — стаб з `Task.sleep` + випадковий success/fail
- Terms та Privacy відкривають реальний браузер через `UIApplication.shared.open`
- Під час покупки екран затемнюється і блокує тапи
